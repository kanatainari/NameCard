# MybusinessCard

SNS Link
Clicking the GitHub or Instagram button opens my SNS page in a web browser.

Profile Image Change
Clicking the "Change Photo" button changes the profile image shown in the PictureBox.

Mouse Effect
When the mouse moves over the photo, the border style of the PictureBox changes to highlight the image.

Background Color Change
Clicking the background color button changes the background color of MyBusinessCard.
